# Sprint Branches

Generate (and remove) temporary branches of each Sprint.

### Usage

```sh
$ npm i
```

Rename the file **config.example.json** to **config.json** and configure it.

```sh
$ node ./sprintBranches.js
```

Enjoy!

### Requirements

- Node >= 10
